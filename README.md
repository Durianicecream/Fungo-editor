# TB Editor

基于 react 和 slate.js 的所见即所得富文本编辑器, 支持文本、图片、链接、emoji 等元素插入

## 在线演示

https://durianicecream.github.io/TBEditor/example/dist/index

## 安装

```
npm install tb-editor --save

cd tb-editor

npm install

npm start
```

### 依赖版本

react ^16.8

slate ~0.34

antd ~3.16
